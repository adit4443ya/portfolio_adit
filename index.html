Fast MIS on Dynamic Graphs

P. Nijhara, Aditya Trivedi, D.S. Banerjee

Proposed novel parallel data structures and algorithmic techniques for efficiently computing Maximal Independent Sets (MIS) on dynamic graphs. Developed specialized approaches to handle batched edge insertions and deletions while maintaining correctness and maximizing parallelism. Achieved 15.64x speedup for insertion operations and 10.57x for deletion operations compared to recomputing from scratch, while maintaining approximately 0.18% cardinality variance. The work introduces innovative conflict resolution strategies that enable efficient parallel MIS maintenance on evolving graph structures, with applications in network analysis, distributed systems, and graph-based machine learning.

Fast MIS on Incremental Graphs

Aditya Trivedi, P. Nijhara, D.S. Banerjee

Introduced novel Tuple Valued Bitmap (TVB) and TVBL data structures that fundamentally improve parallel MIS maintenance for incremental dynamic graphs. These data structures minimize the search space by encoding tuple-level information directly in bitmap representations, enabling efficient identification of affected regions during edge insertions. Achieved 15.64x speedup over static recomputation while successfully processing graphs with over 1.2 billion edges. The TVB/TVBL approach maximizes parallelism by reducing contention and enabling fine-grained concurrent updates, making it suitable for real-time applications requiring frequent graph modifications such as social network analysis and dynamic routing.

ParMIS: Fast and Unified MIS Maintenance Framework for Large Scale Dynamic Graphs

P. Nijhara, Aditya Trivedi, A. H. Singh , Neha Sharma , Avaneesh Pandey, D.S. Banerjee

Presents ParMIS, a comprehensive unified framework for efficient MIS maintenance across all dynamic graph update scenarios including incremental, batch, and fully dynamic modifications. Designed a novel GPU streaming pipeline optimized for billion-scale graphs, achieving 4.63x GPU acceleration and 47.27x batch update speedup over static approaches. Introduces sophisticated conflict resolution strategies that harmonize race-conflicting updates in parallel execution while preserving correctness guarantees. The framework seamlessly handles mixed workloads of insertions and deletions, making it the first unified solution capable of maintaining MIS across diverse dynamic graph scenarios with consistently high performance, applicable to large-scale social networks, knowledge graphs, and streaming graph analytics.

Fast and Accurate MIS on Dynamic Graphs

A. H. Singh, Aditya Trivedi, Neha Sharma, Avaneesh Pandey, D.S. Banerjee

Developed an innovative BFS-based MIS maintenance algorithm that addresses fundamental accuracy challenges in parallel dynamic graph algorithms. The approach intelligently groups race-conflicting updates and executes them in topologically-ordered batches to prevent inconsistencies that plague conventional parallel methods. Achieved 32.4x speedup compared to static recomputation while guaranteeing accurate MIS results without approximation. The BFS-based conflict resolution mechanism identifies dependencies between concurrent updates and schedules their execution to maintain correctness invariants, making it particularly valuable for applications requiring exact solutions such as resource allocation, vertex coloring, and wireless network optimization in dynamic environments.

Semi-External Parallel Maximal Cliques Maintenance

Aditya Trivedi, Dishit Sharma, Aditya Mundhara, D.S. Banerjee

Developing an innovative semi-external memory framework for efficient maximal clique maintenance in billion-scale dynamic graphs that exceed available RAM capacity. The approach strategically partitions the graph between fast in-memory storage and external memory (disk/SSD), achieving 150-300× memory reduction compared to traditional full-graph in-memory methods. Designed specialized data structures and algorithms that minimize I/O overhead while maintaining exact correctness guarantees for maximal clique enumeration. Leveraging GPU parallelism for in-memory computations combined with intelligent prefetching and caching strategies to overlap computation with I/O operations. This work enables processing of graphs previously intractable on single machines, with applications in bioinformatics (protein interaction networks), social network analysis (community detection), and knowledge graph mining.
